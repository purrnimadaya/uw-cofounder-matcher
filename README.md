# 🤝 UW Co-founder Matcher

A lightweight MVP that helps University of Washington students find compatible co-founders based on **commitment, role, trust, and collaboration style**.

## 🚀 Problem
Students struggle to find reliable teammates. Most platforms focus on skills, but real-world collaboration depends on:
- Commitment
- Trust
- Work style
- Follow-through

## 💡 Solution
This app matches students using:
- Role complementarity
- Weekly commitment
- Idea stage (idea → prototype → building)
- Collaboration style
- Trust score (derived from behavioral signals)

## 🧠 Key Insight
> Students care more about commitment and reliability than just skills when choosing teammates.

## ⚙️ Features
- Founder profile creation
- Trust score calculation
- Rule-based matching engine
- Top 3 matches with:
  - Match score
  - Reasons
  - Risks

## 🛠️ Tech Stack
- Python
- Streamlit
- Rule-based matching (v1)

## 🔥 MVP Scope
This is a **2-week sprint MVP** focused on validating:
- What signals matter in co-founder matching
- Whether trust-based matching improves outcomes

## 🚀 Next Steps
- Store user data (Firestore / Sheets)
- Add AI-based matching (Vertex AI)
- Expand beyond UW
